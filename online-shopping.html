Online Shopping Discount
Assingment 2: Restaurant Reservation Eligibility

You are building a web application for an online shopping platform. Write a JavaScript program that prompts the user for their total cart value and their membership status (premium or regular). The program should check if the user is eligible for a discount by applying the following conditions:

  1. If the total cart value is greater than or equal to $100 and the membership status is "premium," the user is eligible for a 20% discount.
  2. If the total cart value is greater than or equal to $50 and the membership status is "regular," the user is eligible for a 10% discount.
  3. If the user does not meet any of the above conditions, they are not eligible for a discount.
